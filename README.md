- 🔭 I’m currently working on **Data Analytics model for forecasting future sales for superstore sales dataset extracted form Kaggle. This project focuses on analyzing and graphically representing the sales from year 2014 to 2017 using graphics library of python, 'Matlab'. The basic comparison gave a clear idea about expected sales versus actual sales for the given tenure. The model used for this analysis is ARIMA (Auto Regressive Integrated Moving Average) based on SARIMAX library. After a clear analysis of all the data, the model was able to successfully forecast future sales for year 2018 and 2019 which surely flags the success of this model. The error rates calculated using MSE and RMSE which came out to be 165859.58 and 407.26 respectively. Results with other analytics tools like LSTM and ByLSTM are expected to produce better results which will come up later.**
